# Relatório de Projeto - JIUConnect

## 1. O Projeto
O JIUConnect é um ecossistema digital para o Jiu-Jitsu Brasileiro, projetado para ser a ferramenta principal de atletas e professores.

## 2. Pilares do Produto
- **Informação:** Busca e centralização de dados sobre a arte suave.
- **Competição:** Calendário global de torneios e resultados.
- **Educação:** Repositório de técnicas e informações para professores.

## 3. Cronograma de Desenvolvimento (Roadmap)

### Fase 1: Fundação (Semana 1)
- Setup do ambiente.
- Definição de Identidade Visual e Marketing.
- Estrutura base do App (Landing Page).

### Fase 2: Funcionalidades Essenciais (Semana 2)
- Dashboard do Atleta.
- Buscador de Torneios.
- Central de Federações.

### Fase 3: Lançamento e Melhorias (Semana 3)
- Implementação de PWA.
- Otimização SEO.
- Lançamento oficial.

## 4. Stack Tecnológica
- **Linguagens:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Design:** Moderno, responsivo e focado em performance.
- **Ferramentas:** AG-Kit para automação e padronização.
